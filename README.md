
# Personal Dotfiles

## General Info

- This repo is a dump for the dotfiles of my favorite programs. 
- You're free to use them. 
- Make sure you know what you're doing before copying things.
 
## Requirements

Must be a bit obvious that you must have the software installed for the dotfiles to work

## Scripts

Run `upload.sh` to update this repo with your system's files

> It is assumed the following system Paths
> `$HOME/.zshrc`, `$HOME/.vimrc`, `$HOME/.vimrc.plug` and `$HOME/.config/kitty/`

Run `download.sh` to update your system with this repo's files
